# EV Charge Estimator ⚡

Una calculadora interactiva para estimar tiempos de carga, energía añadida y costos de un vehículo eléctrico.

## 🚀 Tecnologías
- HTML5
- SASS (modular, con reset y grid)
- JavaScript (módulos: battery.js, charger.js, main.js)

## 📂 Funcionalidades
- Sliders para definir estado actual y objetivo de batería.
- Selección de tipo de cargador (Level 1, Level 2, DC Fast).
- Cálculo automático de tiempo de carga, energía añadida, autonomía y costo estimado.
